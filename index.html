How to play
Every word has roots.
Find them.
The target word is shown broken into colored segments — each color is a root to discover.
Type any word you think shares a root. If it does, that root lights up and you score points.
Gray segments are connective tissue — Latin glue that holds the word together. Not scored, just shown for clarity.
Reveal a root if you're stuck — but finding it yourself first earns a +3 bonus. 🌟
